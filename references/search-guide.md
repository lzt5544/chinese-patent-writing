# 现有技术检索的简要指引

撰写流程中的"检索现有技术"环节，目的是为撰写**背景技术**和**权利要求**提供依据，并预判**新颖性**和**创造性**风险。本 skill 的检索为**简要检索**，不要求穷尽。

## 为什么要检索

1. **撰写背景技术**：需要引用最接近的现有技术，客观指出其缺陷
2. **确定区别技术特征**：明确本发明与现有技术的差异点，这直接影响独立权利要求的"特征部分"
3. **预判新颖性/创造性风险**：如果检索发现完全相同的现有技术，需提醒用户调整技术方案或放弃申请
4. **避免重复发明**：节约用户的时间和申请费用

## 检索策略

**Step 1：构建关键词**

从技术方案中提取：
- **核心技术词**：发明的核心结构/方法/材料（如"真空隔热""TF-IDF""区块链"）
- **上位概念**：核心词的上位分类（如"保温杯"→"容器""隔热装置"）
- **功能/效果词**：解决的问题或达到的效果（如"温度显示""垃圾邮件过滤"）
- **近义词/同义词**：同一技术的不同叫法

**Step 2：确定IPC分类号（可选，简要了解）**

| 领域 | 常见IPC大类 |
|------|------------|
| 机械/结构 | B部（作业；运输）、F部（机械工程） |
| 电学/电子 | H部（电学）、G部（物理） |
| 化学/材料 | C部（化学；冶金） |
| 通信/软件 | H04（电通信技术）、G06（计算；推算；计数） |
| 生物医药 | A部（人类生活必需）、C07/C12（有机化学/生物化学） |

**Step 3：选择检索数据库**

| 数据库 | 网址 | 特点 | 推荐程度 |
|--------|------|------|---------|
| **中国国家知识产权局专利检索系统** | https://pss-system.cponline.cnipa.gov.cn | 官方、免费、数据最全 | ⭐⭐⭐ 首选 |
| **中国及多国专利审查信息查询** | http://cpquery.cnipa.gov.cn | 可查看审查过程文件 | ⭐⭐⭐ 了解审查动态 |
| **智慧芽 PatentSnap** | https://www.zhihuiya.com | 检索体验好，部分功能免费 | ⭐⭐ 辅助 |
| **Google Patents** | https://patents.google.com | 覆盖全球，英文为主 | ⭐⭐ 查外文专利 |
| **佰腾网** | https://www.baiten.cn | 中文界面，免费检索 | ⭐⭐ 入门友好 |

## "够用了"标准

不需要穷尽所有现有技术，达到以下标准即可停止：

- ✅ 找到 **1-3篇最接近的现有技术**（与本发明解决相同或类似问题）
- ✅ 确认了本发明的**核心技术特征**在现有技术中**未被完全公开**
- ✅ 能够清楚描述**本发明与最接近现有技术的区别**
- ✅ 已识别出可能的**新颖性风险点**（如有完全相同的，需提醒用户）

**停止信号：** 已找到2-3篇高度相关的现有技术；或检索1-2个关键词组合后结果已大量重复。

## 检索结果的使用

**1. 背景技术部分**
```
现有技术中，XXX（专利号/文献号）公开了一种...，该方案存在以下缺陷：...
本发明旨在解决上述现有技术中存在的...问题。
```

**2. 独立权利要求的"特征部分"**
- 前序部分：包含与最接近现有技术**共有**的必要技术特征
- 特征部分：用"其特征在于..."写明**区别**于最接近现有技术的必要技术特征

**3. 创造性论述（如有需要）**
- 在背景技术或发明内容中，说明本发明的区别特征产生了**预料不到的技术效果**，或说明区别特征**不是本领域的常规技术手段**

## 检索后的风险提醒

| 检索结果 | 建议 |
|---------|------|
| 未发现完全相同的现有技术 | 可以正常撰写，但提醒用户正式申请前需由代理机构做专业检索 |
| 发现非常接近的现有技术，但本发明有明显区别 | 撰写时重点突出区别特征，在背景技术中引用该现有技术 |
| 发现完全相同的现有技术 | **提醒用户**：该技术方案可能不具备新颖性，建议重新评估申请价值或改进技术方案 |

> **重要提示**：本 skill 的检索为简要检索，**不能替代专业的专利检索（如查新检索、FTO检索）**。正式申请前，建议委托专利代理机构进行专业的新颖性检索。
