# 专利附图草案与配图指南

本指南用于中国发明专利、实用新型专利申请中的附图草案、摘要附图草案、结构示意图、流程图、系统框图和模块图制作。附图草案用于辅助撰写与沟通，正式提交前仍应由专利代理人或制图人员核对。

## 适用边界

- 只在用户明确处于中国专利申请语境时使用，例如“专利附图”“实用新型结构图”“摘要附图”“帮这份专利配图”。
- 不用于普通产品渲染图、电商详情页、营销海报、外观设计图片或与专利申请无关的流程图。
- 实用新型通常必须有附图；发明专利在说明技术方案需要时应配置附图。

## 输出结构

制作附图时，至少输出以下内容：

1. 附图方案：列出图1、图2等每幅图的类型和展示目的。
2. 附图标记表：用“标号 - 名称 - 说明”的结构列出全部部件、模块或步骤。
3. 附图说明：每图一句，符合说明书“附图说明”章节写法。
4. 生图提示词：如果有生图工具，作为调用提示；如果没有生图工具，作为给绘图人员或外部工具的制图说明。
5. 合规提醒：说明草案仅供参考，正式提交前需核对线条、标号、清晰度和文字限制。

## 附图类型选择

| 技术方案 | 推荐附图 | 重点 |
|---------|---------|------|
| 产品结构/实用新型 | 结构示意图、局部放大图、剖视图、爆炸图 | 展示各部件连接关系和区别结构 |
| 机械传动/流体路径 | 剖视图、路径箭头图、局部放大图 | 展示力、运动、流体或热传导路径 |
| 方法/工艺流程 | 流程图 | 用 S1/S2/S3 对应权利要求步骤 |
| 软件/AI/算法 | 系统框图、模块图、数据流图、训练/推理流程图 | 展示模块、数据、模型和处理流程 |
| 控制系统/物联网 | 系统框图、通信拓扑图、控制流程图 | 展示传感器、控制器、执行器和通信关系 |
| 摘要附图 | 最能体现独权核心特征的一幅图 | 简洁，不堆砌实施例细节 |

## 生图工具调用规则

如果当前模型具备生图工具，且用户请求属于中国专利附图/配图语境，必须调用生图工具生成草案图。调用前先形成明确提示词，调用后仍需输出文字版附图标记表和附图说明。

生图提示词应遵守：

- 使用黑白线稿、工程制图、专利附图风格。
- 白色背景，清晰细线，无阴影、无渐变、无材质渲染、无照片质感。
- 不要出现商标、品牌、人物、装饰性背景或营销文案。
- 图中尽量只保留简单数字标号或空白引线；正式标号以附图标记表为准。
- 结构图应强调连接关系、相对位置和关键区别特征，不追求外观美化。
- 流程图和框图可包含必要短词，例如 S1、S2、采集模块、处理模块；避免长句。

## 提示词模板

### 产品结构示意图

黑白线稿专利附图，白色背景，工程制图风格，展示[产品名称]的结构示意图。包括[部件1]、[部件2]、[部件3]，其中[核心区别结构]位于[位置关系]，通过[连接关系]与[相关部件]连接。使用简洁引线和数字标号，线条清晰，无阴影，无渐变，无材质渲染，无品牌标识，无装饰背景。

### 局部放大图/剖视图

黑白线稿专利附图，白色背景，展示[产品名称]中[局部结构]的局部放大剖视图。突出[关键部件]、[层状结构/孔槽/卡扣/密封结构]及其相对位置关系。使用剖面线但避免复杂阴影，使用简洁引线和数字标号，线条清晰，适合专利说明书附图。

### 方法流程图

黑白专利流程图，白色背景，展示[方法名称]的步骤流程。流程从 S1 到 S[末尾]，依次包括[步骤1]、[步骤2]、[步骤3]，必要时包含判断节点[判断条件]和反馈路径[反馈关系]。矩形步骤框和菱形判断框，线条简洁，无颜色渐变，无装饰元素。

### 软件/AI 系统框图

黑白专利系统框图，白色背景，展示[系统名称]。包括[数据采集模块]、[预处理模块]、[模型推理模块]、[结果输出模块]和[存储模块]，用箭头表示数据流。模块为简洁矩形框，标注短词，线条清晰，无图标装饰，无品牌标识。

### 控制系统框图

黑白专利系统框图，白色背景，展示[控制系统名称]。包括传感器、控制器、执行器、通信模块和电源模块，箭头表示信号流或控制流。突出[核心控制关系]，使用简洁数字标号和引线，无阴影、无渐变、无装饰背景。

## 附图标记规则

- 部件标号建议使用 1、2、3 或 10、20、30 等稳定编号，子部件可使用 11、12、21、22。
- 同一部件在全文、附图说明、具体实施方式和附图中必须使用同一标号。
- 权利要求中可在技术特征后用括号标注附图标记，但附图标记不得解释为对权利要求的限制。
- 流程图中的步骤编号应与方法权利要求 S1/S2/S3 保持一致。
- 图中不要出现说明书未提及的标号，说明书也不要引用图中不存在的标号。

## 合规提醒

- 附图应主要使用线条和符号，不应含有照片、效果图、颜色依赖信息或宣传性文字。
- 附图中除流程图、框图必要词语外，不应含有其他注释性文字。
- 附图缩小到 4 厘米 x 6 厘米时仍应能分辨主要细节。
- 生图模型可能生成错误连接、遗漏部件或不稳定标号，交付时必须提醒用户核对。
- 正式提交前应由专利代理人或制图人员根据最终权利要求和说明书统一修订。
